# Points Progression

Source code for the incremental game Points Progression.

## Tech Stack

- React
- TypeScript
- Tailwind CSS

## Requirements

- Node.js
- Git

## Getting Started

```bash
git clone https://github.com/Troxi54/Points-Progression.git
cd Points-Progression
npm install
npm run dev
```

## Scripts

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```
